## The Role of the Project Manager

A project manager is responsible for:
- **Assigning work** to team members    
- **Tracking progress** throughout the project lifecycle    

To effectively manage this, two key terms are used:
- **Project Milestones**    
- **Project Tasks**    

---
## What Is a Project Milestone?

- A **milestone** is a **significant point** in the project schedule.    
- It usually marks the **completion of a deliverable** or a **major phase**.    
- Milestones serve as **progress checkpoints** to ensure the project stays on track.    

### Examples of Milestones
- Completion of the first draft of a report    
- Approval of a major deliverable by the customer or stakeholder    

---
## What Is a Project Task?

- A **task** is a **specific activity** that must be completed within a defined timeframe.    
- Tasks are the building blocks that lead to achieving milestones.    
- Projects are composed of **many tasks**, often performed by different team members.    

### Example: Milestone vs. Tasks

| Milestone                        | Related Tasks                                                        |
| -------------------------------- | -------------------------------------------------------------------- |
| Complete first draft of a report | - Hire a writer  <br>- Conduct research  <br>- Draft report sections |
| Website design approval          | - Create mockups  <br>- Review designs  <br>- Implement feedback     |

---
## Case Study: Project Plant Pals (Office Green)

### Deliverable:

Launch a website for customers to place orders and get support.
### Milestones:

- Approval of the website design    
- Implementation of feedback from user testing    
### Tasks to Reach Milestones:

- Web designer creates mockups    
- Project manager reviews designs    
- Designer incorporates feedback    

All these **tasks** must be completed to reach the **milestones**, which in turn help accomplish the **overall project goals**.

---
## Key Takeaways

- **Milestones**: Key progress markers in the project schedule.    
- **Tasks**: Individual actions required to achieve milestones.    
- Milestones and tasks are **interconnected**:    
    - Tasks **lead up to** milestones        
    - Milestones **signal major achievements**        

Understanding this structure is essential for **tracking progress**, managing timelines, and ensuring project success.
